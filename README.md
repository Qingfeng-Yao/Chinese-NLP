### 如何表示文本+如何进行计算
- 词袋模型(通过统计不同单词的频次来形成文本向量表示)+统计方法
  - 丢失文本的词序信息以及单词之间的联系
- 根据单词的共现学习词向量(语义相似或联系紧密的词在向量空间中距离更小)+深度学习
  - 文本编码：RNN、CNN(将文本类比图像，编码形成矩阵表示)、GNN
- 预训练语言模型(利用自监督学习给单词或文本赋予上下文敏感的、多层的语义向量表示)+transformer
